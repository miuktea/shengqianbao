# 生钱宝（AI模拟理财网页）

生钱宝是一个**零真钱风险**的模拟理财产品，面向理财小白和 Z 世代用户。  
你可以在网页里看到真实股票行情，进行虚拟买卖，获得 AI 风格的新手建议，并用收益模拟器学习长期投资思维。

---

## 1. 项目目标

- 让不懂理财的人快速上手投资基础概念
- 在不投入真钱的情况下体验股票交易流程
- 结合真实市场数据，提升学习的真实性
- 通过可视化交互和界面切换提升体验感

---

## 2. 核心功能

### 2.1 真实行情模块
- 从 Eastmoney 获取美股实时快照（修复 Yahoo 401/429 导致的失败问题）
- 默认展示：`AAPL`、`TSLA`、`MSFT`、`NVDA`、`BABA`
- 展示字段：
  - 股票代码
  - 公司名
  - 当前价格
  - 涨跌额和涨跌幅
  - 交易所与市场状态

接口：
- `GET /api/quotes?symbols=AAPL,TSLA`
- `GET /api/search?keyword=tesla`（搜索股票）
- `GET /api/trends?symbol=AAPL`（分时趋势）

返回示例（简化）：
```json
{
  "updatedAt": 1710000000000,
  "quotes": [
    {
      "symbol": "AAPL",
      "name": "Apple Inc.",
      "price": 211.18,
      "change": -1.22,
      "changePercent": -0.57,
      "currency": "USD",
      "marketState": "REGULAR",
      "exchange": "NasdaqGS"
    }
  ]
}
```

### 2.2 模拟交易模块
- 初始虚拟资金：`100,000 USD`
- 支持买入、卖出、重置账户
- 自动计算：
  - 现金余额
  - 持仓市值
  - 总资产
- 持仓数据保存在浏览器 `localStorage`，刷新页面不丢失

### 2.3 AI新手建议模块（规则型）
- 根据风险偏好（稳健/平衡/进取）生成建议
- 结合当前仓位与市场涨跌面给出学习建议
- 提示“仅教育用途，不构成投资建议”

### 2.4 未来收益模拟器
- 输入假设年化收益率和投资年限
- 基于当前总资产计算理论未来资产
- 用于帮助用户理解长期复利思维

### 2.5 界面体验
- 暗黑/明亮主题切换
- 普通/炫彩界面切换
- Tab 交互页签：
  - 真实行情
  - 模拟交易
  - AI理财建议
  - 学习中心
  - 排行榜

### 2.6 第2版升级能力（已落地）
- 股票搜索 + 自选股（支持中英文关键词，返回真实候选列表，可多选加入）
- 股票分时曲线图（点击卡片切换）
- 每日理财知识卡片 + 新手任务勾选
- 登录系统（昵称）+ 排行榜
- AI理财教练接口（支持接入 OpenAI API；无 Key 时自动规则兜底）
- 多股对比台（最多 6 支股票横向比较）
- 模拟交易升级为“按每笔买入记录”：
  - 记录买入时间、买入价、数量
  - 计算每笔到今天的盈亏和持有天数
  - 支持“今天买、明天回来继续看收益变化”
- 未来收益模拟器已移动到“模拟交易”板块
- AI 建议前置“投资者小测评”（年龄/目标/回撤承受/周期）

## 3. 技术架构（简单易维护）

- 前端：`HTML + CSS + 原生 JavaScript`
- 后端：`Node.js + Express`
- 数据源：Yahoo Finance 公开行情接口（由后端代理请求）

目录结构：
```text
.
├── public
│   ├── app.js
│   ├── index.html
│   └── styles.css
├── package.json
├── README.md
└── server.js
```

---

## 4. 本地运行步骤（新手版）

1) 安装 Node.js（建议 18+）  
2) 在项目目录执行：

```bash
npm install
npm start
```

3) 浏览器打开：`http://localhost:3000`

---

## 4.1 一键对外分享（推荐）

你希望“别人不用安装就能直接体验”，建议部署到 Render：

1) 把项目上传到 GitHub  
2) 打开 [Render](https://render.com/)，新建 `Web Service`，连接该仓库  
3) 配置：
- Build Command: `npm install`
- Start Command: `npm start`
- Node Version: `20`
4) 部署完成后会获得一个公网链接，发给任何人即可直接体验

可选环境变量：
- `OPENAI_API_KEY`：填写后 AI 建议会调用真实大模型

---

## 5. 关键参数说明

### 行情接口参数
- `symbols`：股票代码字符串，多个用逗号分隔（最多 20 个）
- 示例：`/api/quotes?symbols=AAPL,MSFT,TSLA`

### 收益模拟器参数
- 年化收益率：`0 ~ 50`（输入框限制）
- 投资年限：`1 ~ 50`
